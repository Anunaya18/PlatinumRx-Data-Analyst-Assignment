# PlatinumRx-Data-Analyst-Assignment

## 📌 Overview

This project demonstrates core data analysis skills using SQL, Excel, and Python.
It includes database querying, spreadsheet analysis, and basic programming logic.

---

## 🛠 Tech Stack

* SQL (SQLite / MySQL)
* Microsoft Excel / Google Sheets
* Python 3

---

## 📂 Project Structure

* **SQL/** → Database schema and queries for Hotel & Clinic systems
* **Python/** → Scripts for time conversion and string manipulation
* **Spreadsheets/** → Excel file for ticket analysis
* **screenshots/** → Output images for verification
